CORE 104 Research Project: Analysis of 1.88 Million US Wildfires Dataset

About the Dataset

This dataset contains a spatial database of wildfires that occurred in the United States from 1992 to 2015 recordered by the US Department of Agriculture. It includes 1.88 million geo-referenced wildfire records, representing a total of 140 million acres burned during the 24-year period.

Goal

The goal of this analysis is to provide a broad overview of the dataset with a specific emphasis on differences between the US states and changes over time.

Method

I used Python for data processing and cleaning. The visualizations were creating using a combintaion of Python and D3.js. This website was created with HTML, CSS and JavaScript.
